# Bimal Kumal — Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework** — Next.js 14 (App Router)
- **Language** — TypeScript
- **Styling** — Tailwind CSS + CSS custom properties
- **Fonts** — Cabinet Grotesk, Fraunces, DM Mono (Google Fonts)
- **Animations** — CSS keyframes, IntersectionObserver scroll reveal, canvas cursor trail

## Project Structure

```
├── app/                  # Next.js app router (layout, page, globals)
├── components/           # UI components (Nav, Hero, Projects, Skills, etc.)
├── data/                 # Content data (projects, skills, experience)
├── hooks/                # Custom React hooks
└── tailwind.config.ts    # Design tokens (colors, fonts, animations)
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
